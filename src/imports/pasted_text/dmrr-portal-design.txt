Create a **complete modern enterprise-grade Web Portal UI/UX design in Figma** for the **Government of Maharashtra – Disaster Management, Relief & Rehabilitation Department (DMRR)** called:

**“DMRR Project Pipeline Monitoring System (PPMS)”**

This is a **government enterprise portal** used by **vendor manpower, supervisors, PMU, committees, and administrators** for monitoring disaster mitigation proposals and procurement projects across Maharashtra.

## CORE DESIGN REQUIREMENTS

Design a **fully connected clickable portal prototype**, not isolated screens.

Ensure:

* Every page is connected with working navigation.
* Every button navigates logically.
* Every upload, edit, approve, reject, revise, save, submit, download, and view button works.
* Workflow is continuous from Proposal Initiation → DDMA → PMU → PAC → Rationalization → TAC → SEC → SDMA → Tendering → Execution → Billing → Closure.
* Include realistic Maharashtra Government data and naming.
* Use **Maharashtra districts, talukas, departments, and government terminology**.
* Include loading states, empty states, filters, search, notifications, document upload success, errors, and confirmation modals.
* Design must feel premium, modern, aesthetic, and enterprise-grade.
* Avoid blue theme completely.
